## 1.9.6 Gerapy的安装

Gerapy 是一个 Scrapy 分布式管理模块，本节来介绍一下 Gerapy 的安装方式。

## 相关链接

* GitHub：[https://github.com/Gerapy](https://github.com/Gerapy)

## Pip安装

推荐使用 Pip 安装，命令如下：

```
pip3 install gerapy
```

## 测试安装

安装完成之后，可以在 Python 命令行下测试。

```python
$ python3
>>> import gerapy
```

如果没有错误报出，则证明库已经安装好了。
